# Assignment 01 – Student Admission Eligibility Checker

## Topics Covered
- Variables
- Data Types
- User Input & Output
- Operators
- Type Casting

## Business Scenario

A training institute is conducting admissions for a Data Analyst program.

The admission team wants a simple Python application that collects student details and determines whether the student is eligible for admission.

## Requirements

1. Accept:
   - Student Name
   - Age
   - Degree
   - Percentage

2. Display details.

3. Determine eligibility:
   - Percentage >= 60 → Eligible
   - Percentage < 60 → Not Eligible

4. Calculate:
   - Years remaining to reach age 30
   - Percentage required to reach 100
